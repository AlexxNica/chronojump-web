Software Chronojump en OSX El Capitan

Chronojump 1.5.5 y versiones posteriores deberían funcionar correctamente en MacOSX El Capitán.
Esta página contiene instrucciones para restaurar su sistema si usted desactivó csrutil para que funcionase la antigua versión 1.5.4 de Chronojump.
Si usted no desactivó csrutil, entonces no necesita realizar nada de lo que se indica en esta página, simplemente instale Chronojump normalmente.

Reactivar csrutil

  • Reiniciar al Recovery OS: Es necesario reiniciar el ordenador manteniendo pulsadas las teclas Command y R (⌘+R).
  • Abrir un Terminal en el menú de Utilities (o Tools).
  • Escribir este comando: csrutil enable↩
  • Reiniciar el ordenador

Instrucciones ANTIGUAS para Chronojump 1.5.4 en El Capitán (desactivar csrutil)

Instrucciones para la instalación (realizar sólo una vez)
  1. Desactivar csrutil
    • Reiniciar al Recovery OS: Es necesario reiniciar el ordenador manteniendo pulsadas las teclas Command y R (⌘+R).
    • Abrir un Terminal en el menú de Utilities (o Tools).
    • Escribir este comando: csrutil disable↩
    • Reiniciar el ordenador
  2. Desactive Gatekeeper
  3. Instale la última versión de R: Última versión de R para MacOSX
  4. Instale el último driver de Chronopic
  5. Instale Chronojump 1.5.4 en el último paso de la instalación le pedirá si quiere que lo ejecute. No lo haga debido a que no funcionará.
  6. Cree un link a Chronojump
    • Abra el Finder, pulse las teclas: ⌘+⇧+G
    • Escriba esto: /Applications/Chronojump.app/Contents/MacOS/↩
    • Seleccione el archivo "Chronojump" y "Make alias" o "Crear alias" o pulse ⌘+L
    • Se creará un archivo "Chronojump.alias". Muévalo a su escritorio.
Ejecutar Chronojump
Abra el archivo "Chronojump.alias" en su escritorio.
Si tiene cualquier problema, escriba a support@chronojump.org
